# Mockup de Criação de Conta

Este projeto é um mockup de uma página de criação de conta, que inclui um formulário para registro de usuários. Abaixo estão as informações sobre os arquivos e como utilizá-los.

## Estrutura do Projeto

```
mockup-criacao-conta
├── src
│   ├── index.html        # Página HTML principal com o formulário de criação de conta
│   ├── css
│   │   └── signup.css    # Estilos CSS para a página de criação de conta
│   └── js
│       └── main.js       # Scripts JavaScript para interatividade
├── .gitignore            # Arquivos e diretórios a serem ignorados pelo Git
└── README.md             # Documentação do projeto
```

## Instalação

1. Clone o repositório:
   ```
   git clone <URL_DO_REPOSITORIO>
   ```

2. Navegue até o diretório do projeto:
   ```
   cd mockup-criacao-conta
   ```

3. Abra o arquivo `src/index.html` em um navegador para visualizar o mockup.

## Uso

- O formulário permite que os usuários insiram informações como nome, e-mail e senha.
- O CSS em `src/css/signup.css` estiliza a página para uma melhor apresentação.
- O arquivo `src/js/main.js` pode ser utilizado para adicionar funcionalidades interativas, como validação de formulários.
